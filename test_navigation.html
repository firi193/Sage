Sage UI Navigation Test

Manual Tests

Open the main Sage UI and test the following:

  • Click on each navigation tab (Keys, Grants, Logs)
  • Use keyboard navigation (Tab to focus, Arrow keys to move, Enter/Space to activate)
  • Use Alt+1, Alt+2, Alt+3 shortcuts
  • Check URL hash updates when switching views
  • Test responsive behavior on mobile screen sizes
  • Verify ARIA attributes are present for screen readers
